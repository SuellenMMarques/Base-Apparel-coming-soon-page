# 🛍️ Base Apparel Coming Soon

Projeto desenvolvido como solução do desafio do Frontend Mentor, com foco em HTML, CSS e JavaScript para criação de uma landing page responsiva com validação de formulário.

---

## 📌 Sobre o projeto

Este projeto consiste em uma página de “coming soon” para uma loja de moda, com layout responsivo e validação de e-mail no formulário.

O principal objetivo foi praticar:

* Estruturação semântica com HTML
* Estilização responsiva com CSS (Mobile First + Grid)
* Manipulação do DOM com JavaScript
* Validação de formulário

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Funcionalidades

* Layout totalmente responsivo (mobile, tablet e desktop)
* Troca de imagem com `<picture>` para diferentes resoluções
* Validação de e-mail com JavaScript
* Exibição de mensagem de erro
* Feedback visual no input (ícone de erro)

---

## ⚙️ Como funciona a validação

* O formulário captura o valor digitado pelo usuário
* Verifica se o campo está vazio ou inválido
* Caso esteja:

  * Exibe mensagem de erro
  * Adiciona estilo visual ao input
* Caso esteja válido:

  * Remove a mensagem de erro
  * Limpa o estado visual

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando abordagem **Mobile First**, com adaptações para:

* 📱 Mobile
* 📲 Tablet
* 💻 Desktop (uso de CSS Grid para layout)

---

## ♿ Acessibilidade

* Uso de `label` com classe `sr-only` para leitores de tela
* Controle de erro com feedback visual no campo de input
